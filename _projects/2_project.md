Built a machine learning model to predict sepsis at early stages using ICU data from the 2019 PhysioNet Challenge.

**Highlights**

- Handled missing values in time-series clinical data using linear interpolation and MICE imputation.
- Addressed class imbalance and trained logistic regression and XGBoost models.
- Optimized for recall to reduce false negatives, reaching up to 80% recall.
